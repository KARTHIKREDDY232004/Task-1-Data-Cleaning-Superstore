# Task 1: Data Cleaning and Preprocessing

## Objective

The objective of this task was to clean and prepare a raw dataset for analysis by identifying data quality issues and applying appropriate preprocessing techniques using Microsoft Excel.

## Dataset

* Dataset Name: Sample Superstore
* Records: 9,994
* Tool Used: Microsoft Excel

## Data Cleaning Steps Performed

### 1. Missing Value Check

* Checked all columns for missing values using Excel filters.
* Filled them with appropriate values.

### 2. Duplicate Removal

* Identified duplicate records using Excel's Remove Duplicates feature.
* Removed duplicate rows to improve data quality.

### 3. Text Standardization

* Reviewed categorical columns such as Ship Mode, Segment, Region, Category, and Sub-Category.
* Checked for inconsistencies and unnecessary spaces.
* Standardized text formatting where required.

### 4. Column Header Cleaning

* Renamed column headers using a consistent naming convention.
* Converted headers to lowercase and replaced spaces with underscores where appropriate.

### 5. Data Type Validation

* Verified numerical columns including Sales, Quantity, Discount, and Profit.
* Ensured data was stored in the correct format for analysis.

### 6. Data Quality Validation

* Reviewed the dataset for consistency and accuracy.
* Confirmed that the cleaned dataset was ready for further analysis and reporting.

## Outcome

The dataset was successfully cleaned and validated. Duplicate records were removed, column names were standardized, data types were verified, and overall data quality was improved. The final dataset is ready for analysis, visualization, and business reporting.
